Pointer arithmetic

int pointers are incremented or decremented by sizeof(int)
bytes. void pointers are incremented or decremented by 1 byte
since sizeof(void) is illegal.

char *my_strcpy(char dest[], char src[]) {
int i = 0;
while (src[i] != \0) {
dest[i] = src[i];
i++;
}
dest[i] = \0;
return dest;
}

When to use pointers?
Indirect addressing

Pointers vs. references
Pointers may be NULL
References have to be valid (but may not if misused)
As parameters, small objects should behave like ints, e.g.
std::string.
Resource management
Must NOT have memory leaks
Acquiring and releasing tend to behave in a well-nested fashion
Across the borders of functions/methods, use smart pointers
